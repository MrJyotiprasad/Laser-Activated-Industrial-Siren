# Laser Based Industrial Siren System

An Arduino-based industrial siren control system with OLED graphical interface, adjustable delay timer, trigger detection, and countdown monitoring.

## Features

- OLED Animated UI
- Adjustable Siren Delay
- Trigger Detection
- Countdown Popup
- Analog Delay Control
- Boot Animation
- Industrial Interface Design

## Hardware Used

- Arduino UNO/Nano
- SSD1306 OLED Display
- Push Button / Laser Trigger
- Potentiometer
- Relay / Siren Driver

## Libraries

- Adafruit GFX
- Adafruit SSD1306
